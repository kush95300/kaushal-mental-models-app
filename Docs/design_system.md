# UI/UX Design System & Checklist

## Design Principles

1.  **Premium Aesthetic**: Use rich colors, glassmorphism, and smooth animations.
2.  **Dynamic Interactions**: Hover effects, micro-interactions, and fluid transitions.
3.  **Visual Hierarchy**: Clear distinction between Urgent (Red), Scheduled (Blue), Delegated (Amber), and Eliminated (Gray).

## UI Checklist

### Layout & Structure

- [ ] **Responsive Design**: Works perfectly on Mobile, Tablet, and Desktop.
- [ ] **Container Widths**: Max-width constraints for readability on large screens.
- [ ] **Grid System**: Consistent spacing (4px, 8px, 16px, 24px, 32px).

### Components

- [ ] **Buttons**:
  - Primary: Gradient background, shadow, scaling on hover.
  - Secondary: Outline or ghost with subtle hover bg.
  - Danger: Red/Rose with confirmation steps.
- [ ] **Cards (Tasks)**:
  - Glassmorphism effect (backdrop-blur).
  - Draggable handle.
  - Clear status indicators.
- [ ] **Modals**:
  - Centered, backdrop blur overlay.
  - Smooth entry/exit animations.
  - Clear "Close" and "Confirm" actions.

### Typography

- [ ] **Headings**: Bold/Black weight, tight tracking.
- [ ] **Body**: Readable font size (min 14px), good contrast (slate-500/slate-900).
- [ ] **Labels**: Uppercase, tracking-widest, small text for metadata.

### Color Palette

- **Primary (Action)**: Indigo/Purple Gradients.
- **Urgent (Do)**: Rose-500.
- **Schedule**: Indigo-500.
- **Delegate**: Amber-500.
- **Eliminate**: Slate-500.
- **Background**: Slate-50 / Slate-900 (Dark Mode).

### New Features UI

- [ ] **Workspace Switcher**: Dropdown or Sidebar tab in the header.
- [ ] **Analytics Graphs**: Clean, minimal charts (Recharts or similar).
- [ ] **Calendar View**: Grid view with task pills.
